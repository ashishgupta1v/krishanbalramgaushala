==================================================
Krishan Balram Gaushala - Custom Audio Directions
==================================================

To use a custom bhajan or audio file when users open this application:

1. Rename your audio file to `bhajan.mp3`
2. Place the file inside this directory:
   `public/audio/bhajan.mp3`

Our application is built to load this file automatically. If the file is missing or fails to load, it will automatically fall back to a serene, copyright-free instrumental bansuri/flute melody from a public CDN.
